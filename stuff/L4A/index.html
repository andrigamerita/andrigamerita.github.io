L4A - Linux4Android

A collection of scripts and resources to install and use native full-fleged Linux on top of Android.


Protip: Click on the button on the right-bottom of the screen to switch between light and dark theme.

Note: Javascript needs to be enabled on this page for clipboard-copying buttons and color theme switching.

Prerequisites

First thing, to use this you need to have a Linux subsystem (like Termux) installed on your Android device, where you will paste the various commands.


If you also want to use a Desktop Environment or just a GUI of any sort, you need to connect to the VNC server (with address: localhost) started from the Linux environment. To achieve this, you need an app like VNC Viewer.
